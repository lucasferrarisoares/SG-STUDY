Bem-vindo ao Hospital SG

Este é o sistema de gestão hospitalar.
Utilize o menu acima para acessar as funcionalidades principais:

  • Quartos: Gerencie os quartos disponíveis no hospital.
  • Camas: Controle as camas e sua ocupação.
  • Pacientes: Cadastre e acompanhe os pacientes.
  • Hospital: Cadastre e visualize hospitais.

Selecione uma opção no menu para começar.
